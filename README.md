gitstat
======

## Usage

### commitmonth.py

    [root@localhost gitstat]# ./commitmonth.py -h
    Usage: commitmonth.py [options]

    Options:
      -h, --help            show this help message and exit
      --after=YYYY-MM(-DD)  
      --before=YYYY-MM(-DD)
      --sort                
      --graph               requires ioctl(TIOCGWINSZ)
      --branch=BRANCH
